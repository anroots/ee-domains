.ee domains

This is a plain-text list of all .ee internet domains.

The list is compiled from the .ee AXFR zone file once a day. It does not include any subdomains. The primary purpose of the list is to help (ethical) internet researchers obtain a list of domains.

List

  • lists/domains.txt - full list of all .ee domains (~6MB)
  • lists/last-update.txt - UNIX timestamp on when the lists were last updated
  • lists/added.txt - list of new domains that were added during the last update
  • lists/deleted.txt - list of removed domains that deleted during the last update

All of the above files are also available in JSON format - just change the file extension from .txt to .json.

Usage

You can download the lists directly using cURL or any other client. As a matter of courtesy, please include a descriptive User-Agent header when doing this repeatedly.
