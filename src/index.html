.ee domains

This is a plain-text list of all .ee internet domains.

The list is compiled from the .ee AXFR zone file once a day. It does not include any subdomains. The primary purpose of the list is to help (ethical) internet researchers obtain a "clean" list of domains in a format suitable for immediate use.

You can download the lists directly using cURL or any other client. As a matter of courtesy, please include a descriptive User-Agent header when doing this repeatedly. Don't DoS this server. There are no uptime / API freeze guarantees.
